# Download Instructions
## Amazon Catalog Price Dataset

The dataset can be found on [Kaggle](https://www.kaggle.com/datasets/debarghamitraroy/amazon-catalog-price-dataset?resource=download).

## Dataset files

Expected files:
- `train.csv`: training data with the target variable `price`
- `test.csv`: test data for final evaluation
- `sample_test.csv`: sample test input
- `sample_test_out.csv`: sample output for `sample_test.csv`

## Data description

Main columns:
- `sample_id`: unique identifier for each sample
- `catalog_content`: product title, description, and item pack quantity
- `image_link`: public URL for the product image
- `price`: target variable, available only in training data
